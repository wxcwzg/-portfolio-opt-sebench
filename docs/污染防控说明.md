## 污染防控说明

### 是否允许联网
不允许。所有数据已在本地提供，Python依赖预装在Docker环境中。

### 防止Agent直接搬运现有代码
1. 数据是合成生成的（因子模型+特定随机种子），不对应任何公开数据集，无法直接套用现成方案
2. 测试集收益率对Agent完全不可见（仅在Judge容器中）
3. 数据中人为加入了regime change和均值反转效应，直接套用教科书公式会表现很差
4. 提交物仅为weights.csv（权重矩阵），不是代码——Agent必须真正运行优化才能生成有效权重

### LLM训练数据覆盖可能性
投资组合优化是经典问题，LLM训练数据中包含大量相关教材和代码。但影响有限：
1. 本任务的数据是独特的合成数据，具有特定的因子结构和regime特征，记忆通用方法不能直接拿高分
2. 评分是连续的（夏普比率），能精细区分优化深度
3. 真正的高分需要针对本数据特点的精细调参和方法组合，这不可能被记忆

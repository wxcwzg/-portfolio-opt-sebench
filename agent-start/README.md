# 投资组合动态优化

## 项目概述

你是一名量化研究员，需要为一个包含50只资产的投资组合构建动态优化系统。给定4年的历史日收益率数据（训练集），你需要生成未来1年（250个交易日）每天的最优投资组合权重，最大化样本外的风险调整后收益（夏普比率）。

## 数据

- `data/returns_train.csv`: 训练集，1000个交易日 × 50只资产的日收益率
  - 索引为日期，列为资产名称（ASSET_00 ~ ASSET_49）
  - 数值为日收益率（如0.01表示当日涨1%）

## 任务要求

生成文件 `weights.csv`，格式：
```
date,ASSET_00,ASSET_01,...,ASSET_49
2023-11-02,0.02,0.03,...,0.01
2023-11-05,0.025,0.028,...,0.012
...
```

每行为一个交易日的组合权重。测试期日期为训练集最后一天之后的250个交易日。

## 约束条件

1. **权重之和 = 1**（允许±5%误差）
2. **所有权重 >= 0**（仅做多，不允许做空）
3. **交易成本**：每次调仓按换手率收取单边0.1%费用
4. 不允许联网，仅CPU，内存≤4GB

## 评分方式

```
总分 = min(100, sharpe_ratio × 10)
```

夏普比率 = 年化收益率 / 年化波动率（扣除交易成本后）

约束违规（权重不归一或存在做空）会导致分数减半。

## 提示

- 等权组合（1/N策略）是最简单的baseline，但不是最优的
- 训练集中的统计特征（均值、协方差）在测试期可能发生变化（regime change）
- 协方差矩阵的估计质量直接影响优化结果
- 考虑：收缩估计、风险平价、最小方差、Black-Litterman、动态再平衡等方法
- 过度优化训练集可能导致样本外表现很差（过拟合风险）
